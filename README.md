# GRUFON Platform

Official platform repository of the ERENZO Association.

GRUFON is a research-oriented platform focused on the reporting, documentation, mapping, and structured analysis of Unidentified Anomalous Phenomena (UAP).

## Scope

- UAP reporting workflows
- Mapping and visualization
- Case documentation
- Research support tools
- Platform integrations
